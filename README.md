# Food Ordering App - Backend

A RESTful backend API built with Node.js, Express.js, and MongoDB Atlas for managing food items and application data. The backend provides CRUD operations, dashboard statistics, and database integration for the Food Ordering Application.

## Features

* REST API Development
* CRUD Operations
* MongoDB Atlas Integration
* Mongoose Models
* Dashboard Statistics API
* Express.js Routing

## Tech Stack

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose

## Deployment

* Backend: Render

## API Endpoints

### Foods

* GET /foods
* POST /foods
* PUT /foods/:id
* DELETE /foods/:id

### Dashboard Statistics

* GET /stats
